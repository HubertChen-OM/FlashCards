About Us

Hello, my name is Hubert Chen and I am the creator of this web application. My favorite method of studying is making flashcards. However, there is some overhead involved in this study method (e.g., cost, portability, tedious). A lot of these disadvantages can be overcome with the help of technology. I hope that this application helps you in studying your material.

One of the main reasons I chose to pursue this idea is because of the current alternatives. A lot of competitiors/alternatives require you to pay extra for features and do not provide a satisfying user experience. I have made this site open source so that others may contribute to improving it. Please feel free to contact me if you have any suggestions for the application.
